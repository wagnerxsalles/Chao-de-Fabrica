 Sistema de Controle de Produção - Fábrica de Peças Íntimas

Este projeto foi desenvolvido para resolver um problema real de controle de produção em uma fábrica de peças íntimas, permitindo acompanhar o desempenho dos funcionários, tempo de serviço e produtividade geral da produção.

---

 Objetivo do Sistema

O sistema tem como objetivo:

- Controlar o tempo de serviço de cada funcionário**
- Monitorar quantas pessoas estão trabalhando no momento**
- Calcular a produção individual por funcionário**
- Calcular a produção total da fábrica**
- Separar a produção por **código de peça**
- Medir a porcentagem de produtividade geral**

---

 Como o sistema funciona

Cada funcionário registra sua produção informando:

- Nome do colaborador
- Código da peça produzida
- Quantidade produzida
- Tempo de trabalho

O sistema automaticamente:

- Soma a produção individual
- Agrupa por código de peça
- Calcula o total geral produzido
- Calcula a produtividade percentual da fábrica
- Conta quantos funcionários estão ativos no momento

---

 Principais funcionalidades

-  Cadastro de produção por funcionário  
-  Controle de funcionários ativos  
-  Cálculo de tempo de serviço  
-  Cálculo automático de produção total  
- Percentual de produtividade geral  
-  Separação por código de peça  
- Relatórios de produção individual e geral
